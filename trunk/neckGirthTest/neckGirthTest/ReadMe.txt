================================================================================
    MFC 程式庫：neckGirthTest 專案概觀
===============================================================================

應用程式精靈已經為您建立了這個 neckGirthTest 應用程式。這個應用程式不僅示範了使用 MFC 的基本概念，也是您撰寫應用程式的起點。

這個檔案包含一份摘要，簡要說明構成 neckGirthTest 應用程式的
所有檔案。

neckGirthTest.vcproj
    這是使用應用程式精靈所產生之 VC++ 專案的主要專案檔。
    它含有產生該檔案之 Visual C++ 的版本資訊，以及有關使用應用程式精靈所選取之平台、組態和專案功能的資訊。

neckGirthTest.h
    這是應用程式的主要標頭檔。它包含其他專案特定的標頭 (包括 Resource.h)，並宣告 CneckGirthTestApp 應用程式類別。

neckGirthTest.cpp
    這是主要的應用程式原始程式檔，它包含應用程式類別 CneckGirthTestApp。

neckGirthTest.rc
    這是程式所用的所有 Microsoft Windows 資源的列表。它含有儲存在 RES 子目錄中的圖示、點陣圖和游標。您可以直接在
    Microsoft Visual C++ 內編輯這個檔案。您的專案資源在 1028。

res\neckGirthTest.ico
    這個圖示檔是用來做為應用程式的圖示。這個圖示包含在主要資源檔 neckGirthTest.rc 內。

res\neckGirthTest.rc2
    這個檔案包含 Microsoft Visual C++ 編輯無法編輯的資源。您應該將所有資源編輯器無法編輯的資源放到這個檔案。

/////////////////////////////////////////////////////////////////////////////

主框架視窗:
    專案含有標準 MFC 介面。

MainFrm.h, MainFrm.cpp
    這些檔案包含框架類別 CMainFrame，該框架類別衍生自
    CFrameWnd，並控制所有 SDI 框架功能。

res\Toolbar.bmp
    這個點陣圖檔是用來建立並排顯示在工具列的影像。
    初始工具列和狀態列是在 CMainFrame
 類別中建構的。您可以使用資源編輯器，編輯這個工具列點陣圖，而且，您也可以更新 neckGirthTest.rc 的 IDR_MAINFRAME TOOLBAR 陣列，即可新增工具列的按鈕。
/////////////////////////////////////////////////////////////////////////////

應用程式精靈建立文件類型和檢視:

neckGirthTestDoc.h、neckGirthTestDoc.cpp - 文件
    這些檔案包含您的 CneckGirthTestDoc 類別。編輯這些檔案，即可新增您的特殊文件資料，並實作檔案儲存和載入 (透過 CneckGirthTestDoc::Serialize)。

neckGirthTestView.h、neckGirthTestView.cpp - 文件的檢視
    這些檔案包含您的 CneckGirthTestView 類別。
    CneckGirthTestView 物件是用來檢視 CneckGirthTestDoc 物件。





/////////////////////////////////////////////////////////////////////////////

其他功能:

ActiveX 控制項
    應用程式支援 ActiveX 控制項的使用。

列印和預覽列印支援
    應用程式精靈從 MFC 程式庫呼叫 CView 類別中的成員函式，產生了可以處理列印、列印設定和預覽列印命令的程式碼。

/////////////////////////////////////////////////////////////////////////////

其他標準檔案：

StdAfx.h, StdAfx.cpp
    這些檔案是用來建置名為 neckGirthTest.pch 的先行編譯標頭 (PCH) 檔，
    以及名為 StdAfx.obj 的先行編譯型別檔。

Resource.h
    這是標準標頭檔，它定義新的資源 ID。
        Microsoft Visual C++ 會讀取和更新這個檔案。

neckGirthTest.manifest
	應用程式資訊清單檔案是 Windows XP 用來在並存組件的特定版本上，描述應用程式的相依性。載入器會利用這項資訊，從組件快取載入適當的組件，或者從應用程式載入私密金鑰。應用程式資訊清單可能用來做為外部 .manifest 檔的轉散發，	這個 .manifest 檔的安裝位置，和應用程式可執行檔的資料夾相同，或者，它也可能以資源的形式包含在可執行檔內。

其他注意事項:

應用程式精靈使用 "TODO:" 來指示您應該加入或自訂的原始程式碼部分。

如果您的應用程式使用共用 DLL 中的 MFC，您將需要轉散發 MFC DLL。如果您的應用程式與作業系統的地區設定不同，您也必須轉散發對應的當地語系化資源 MFC90XXX.DLL。
如果需要這些主題的詳細資訊，請查閱 MSDN 文件有關轉散發 Visual C++ 應用程式的章節。
